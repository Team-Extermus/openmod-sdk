# OPEN MOD - A OPEN SOURCE FPS SDK

[TOC]

## ABOUT

OpenMod SDK Is a Mod Template built over MapBase, the goal of this mod is allow mod makers make their own mod with some nice perks.

## BUILDING

For building please refer to this guide: [Source SDK 2013 Setting up (Platform Specific)](https://developer.valvesoftware.com/wiki/Source_SDK_2013#Setting_up_.28Platform_Specific.29)

Currently supported platforms:

* Windows
* Linux (WIP)
* Mac (Not Tested)
